App.js

The App.js file is the entry point for our application. It imports some of the necessary modules to help our application work off the bat.

It also creates the SQLite tables needed in order to store our table data. The SQLite module is used under a library called expo-sqlite-query-helper

Create Table

Async function to create new table. under the hood it runs CREATE TABLE IF NOT EXIST.

import { createTable } from 'expo-sqlite-query-helper';

createTable(tableName: string, columns: { [key: string]: string });

Insert

Async function to run insert data into the table, Takes array of objects to insert into specified table. under the hood it runs INSERT INTO table (...columns{keys}) values ...(columns{values});

import { insert } from 'expo-sqlite-query-helper';

insert(table: string, data: InsertObject[]);

tableName - Name of the table to insert data.
data - array of objects to insert into table. example: [{name:"test1",email:"test1@emmail.com"},{name:"test2",email:"test2@exmail.com"}]. Return promise resolving with rowsAffected, insertId, lastQuery

getSubtopics(topic_id) {
    let data = {
      subtopic_id: topic_id
    }
    return this.http.post(this.api_url + '/api/steamSubtopics', data)
      .pipe(retry(1),
        catchError(this.handleError)
      );
  }
